(Программный проект) 

Тема: Детектирование аномалий в данных 

Студент: Манташев Асадулла Уллубиевич


Сервис для детектирования мошеннических финансовых транзакций на основе датасета IEEE-CIS Fraud Detection.



---

## Основной функционал

- загрузка CSV-файла с транзакциями;
- preprocessing входных данных;
- inference ML-моделей;
- fraud probability prediction;
- SHAP explainability;
- REST API;
- Docker Compose deployment.

---

- SHAP
- Docker / Docker Compose
- pandas / numpy
